Homework Problem 6-37
---------------------

int array_t[N][N]
assuming sizeof(int) = 4

CACHE CONFIGURATION
Cache size = 4096 bytes
Block size = 16 bytes
Cache sets = 256 sets
Block bits = log2(16) = 4 bits
Set bits = log2(256) = 8 bits

For N = 64, size of array_t = 64*64*4 = 16384 bytes
For N = 60, size of array_t = 60*60*4 = 14400 bytes
start address = 0x08000000

For block size of 16, four contigous array elements can be held in single cache set.

63 * 4 = 255 bytes
60 = 0x3C
